Used Cars Analysis in Gulf Area
This project analyzes used car market data in the Gulf area using Power BI, focusing on pricing , performance,
and usage patterns based on fuel type, transmission, cylinders , and brand,

The project includes data cleaning, transformation, and interactive dashboard creation with multi-page navigation for deeper data exploration.

Project Objectives:
Analyze used car prices
Compare performance metrics (Top Speed & Torque)

Study the impact of :
- Fuel type
- Transmission type
- Number of cylinders
- Area of use
- Analyze mileage distribution by gear type
- Enable interactive data exploration

  Tools Used
  Power BI
  Data Import & Cleaning
  Data Transformation
  Built-in Aggregations(Count , Sum , Average)
  Multi-page Dashboard Design
  Filters,Slicers, and Navigation Buttons

  Key Insights:
  Automatic transmission vehicles dominate the market
  Higher cylinder counts are associated with higher top speeds
  Petrol vehicles are the most common fuel type
  Vehicle price very signigicantly by brand and transmission

  Notes:
  The dataset contains mixed-language categories .
  No renaming was applied in order to maintain the original data structure.
